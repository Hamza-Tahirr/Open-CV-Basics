# Open-CV-Basics

In this Repo I have covered beginner level examples of OpenCV and build a mini project on cropping tool using basic commands.

## Basics Includes 

- Reading an Image
- Showing an Image
- Converting color pics into grayscale and viceversa
- Playing with RGB
- Resizing the Image 
- Flipping the Image
- Cropping an Image 
- Saving an Image
- Drawing Shapes 
- Live Direct Drawings and Events
- Working with Videos.
